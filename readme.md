# Import for Java ( Imp4j )
也许是最好用的文件直导入库工具

> * 支持xls、xlsx、csv格式
> * 通过配置json文件完成导入逻辑
> * 支持简单公式
> * 支持简单数据校验

### 依赖说明
* [表达式引擎aviator主页](https://github.com/killme2008/aviator)
